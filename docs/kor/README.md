# 한국어 사용 가이드

원자력 제어 멀티에이전트 레드티밍 시뮬레이터의 한국어 가이드 모음입니다.

## 시작하기 전에

본 시스템은 **격리된 mock 환경**입니다 — 실제 원전 시스템에 연결되지 않으며, 모든 변수·알람·계통 정보는 공개 자료 기반의 추상화입니다.

## 시스템 개요

```
[외부 어택 도구]──HTTP/REST──► localhost:8080
                                    │
                                    ▼
                          ┌────────────────────┐
                          │ simulator 컨테이너  │
                          │ - 운전팀 5명        │
                          │ - 플랜트 시뮬       │
                          │ - 가드레일 G0~G10  │
                          │ - mock human       │
                          └────────────────────┘
                                    │
                                    ▼
                          ┌────────────────────┐
                          │ llm_proxy 컨테이너  │
                          │ Anthropic/OpenAI/  │
                          │ Google/Qwen 허용   │
                          └────────────────────┘
```
